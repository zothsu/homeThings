Thank you for purchasing SOLO+ DIY drawings set!
Hw to use files in this folder:
1. PDF file contains drawings, cut lists, sections and step-by-step assembly guide for getting the details ready and building the cabin
2.XLSX file contains spreadsheet with material specifications and a shopping list with possibility to add unit prices so you can get total cost for materials calculated quickly.
3. DWG file with floorplans in case if you need to have your architect placing the cabin on your plot plan. 
4. SKP file can be used to place furniture and walk through the 3D cabin. You need Trimble SketchUp 3D software to open this file. Please see www.sketchup.com. They do have free, online version as well.
